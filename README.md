# Plus30

Currently on development!

Plus30 aims to automate some tasks for the custom game lobbies (Centered around Howling Abyss).



# Planned Features

- Auto Invite
- Whitelist/Blacklist
- Lobby Filtering
- Custom Stats for Custom Games
- Discord Integration (?)
- Custom Ban/Pick System For Champion Selection (?)
